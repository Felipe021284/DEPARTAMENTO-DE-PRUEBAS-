Y
CARGAR TESTPLAN

pasos para instalacion

abrir ventana de bt basic

escribir el comando teshead 1 y precionar enter para habilitar la cabeza de pruebas

escribir el comando fix lock,precionar enter y presionar el pedal para anclar el fixture

escribir el comando msi'ruta del testplan' y precionar enter
para direccionar a la carpeta del testplan

escribir el comando cat y precionar enter
para mostrar los archivos de la carpeta y validar que este el archivo del testplan

escribir el comando get"testplan" y precionar enter
para cargar el testplan

escribir el comando load board y precionar enter
para cargar parametros

escribir el clear nrun y precionar enter
para limpiar errores guardados en corrida anterior

escribir run y precionar enter
para iniciar la prueba

nota

algunos modelos solicitan revicion antes de iniciar la prueba
