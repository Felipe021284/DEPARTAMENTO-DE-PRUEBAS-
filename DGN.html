COMO USAR
DGN

EL DGN SE UTILIZA PARA REALIZAR UN DIAGNOSTICO DEL EQUIPO ICT Y VALIDAR ERRORES PARA REALIZAR SEGUIR LOS SIGUIENTES PASOS

parar la prueba con la opcion de stop

escribir el comando fix unlock, dar enter con el teclado para y presionar el pedar para retirar el fixture

proceder abrir la aplicacion DGN

en la ventana principal seleccionar la opcion system diagnostics


seleccionar la opcion de pin card relay test para verificar errores en pincard

pisar el pedal para iniciar la prueba

terminada la prueba si existen errores se mostraran en la ventana

para una mayor validacion es necesario correr una validacion de pincard relays


seleccionar la opcion de confirmation para verificar errores en pincard

pisar el pedal para iniciar la prueba

terminada la prueba si existen errores se mostraran en la ventana

si no se muestran errores pero el equipo continua fallando es nesesario correr el full diagnostics para validar la control card y la asru card


seleccionar la opcion de full diagnostics para verificar errores en control card y asru

pisar el pedal para iniciar la prueba

terminada la prueba si existen errores se mostraran en la ventana
