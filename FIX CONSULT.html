COMO USAR
FIX CONSULT

para abrir la aplicacion es necesario parar la prueba con la obcion de stop de la pantalla de basic

escribir el comando fix consult,dar enter y esperar un momento

una vez que aparezca todas las ventanas hacer click en la opcion de yes para cargar los parametros

una vez cargados los paranetros se mostrara la ventana principal

para buscar un pin es necesario saber el nombre del nodo para eso es necesario obtener la informacion por medio de la aplicacion de board graph buscando los nodos del componente con search device

escribir el numero de componente y hacer click en show data device paga mostrar la informacion de los nodos del componente

se abrira una ventana que muestra la informaciom de los nodos del componente copiar el nombre del nodo y buscar en fix consult

en la ventana de fix consult utilizar la opcion de busqueda search for node

pegar el nombre del nodo optenido de la aplicacion board graph y hacer click en locate

se abrira una ventana que mostrara el pin selecionado en otro color

se puede utilizar el zoom para acercar o alejar la ventana para una mayor presicion de busqueda y utilizar la ventana auxiliar para ver la referencia completa
